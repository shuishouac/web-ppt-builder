# Theme Presets

Use these presets before writing the deck.

## `editorial-light`

- Fit:
  - company overview
  - consulting deck
  - internal strategy brief
  - investor-style clarity without startup neon
- Mood:
  - clean
  - bright
  - structured
  - credible
- Motion:
  - restrained
  - fade and stagger only

## `product-grid`

- Fit:
  - SaaS demo
  - product feature launch
  - workflow explainer
  - tool comparison
- Mood:
  - modern
  - systematic
  - modular
  - screen-first
- Motion:
  - click reveals
  - lateral transitions
  - UI card emphasis

## `midnight-stage`

- Fit:
  - keynote-style founder deck
  - premium product launch
  - high-contrast talk
  - event presentation
- Mood:
  - dark
  - premium
  - dramatic
  - high-contrast
- Motion:
  - strong transitions
  - spotlight reveals
  - limited but deliberate movement

## `warm-story`

- Fit:
  - creator talk
  - educational narrative
  - case study
  - personal brand presentation
- Mood:
  - warm
  - human
  - lightly editorial
  - approachable
- Motion:
  - soft reveal
  - low-pressure transitions

## Theme Decision Rule

If the user cannot choose:

1. pick `product-grid` for anything tool- or product-driven
2. pick `editorial-light` for business clarity
3. pick `midnight-stage` for impact-led keynote framing
4. pick `warm-story` for personality-led narrative

## Mode Pairing

Recommended pairings:

- `content-dense` + `editorial-light`
- `content-dense` + `product-grid`
- `design-led` + `midnight-stage`
- `design-led` + `warm-story`
